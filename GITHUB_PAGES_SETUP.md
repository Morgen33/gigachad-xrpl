# GitHub Pages Setup Instructions

## Enable GitHub Pages:

1. Go to your repository: https://github.com/Morgen33/gigachad-xrpl
2. Click on **Settings** (top menu)
3. Scroll down to **Pages** (left sidebar)
4. Under **Source**, select:
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Click **Save**
6. Wait 1-2 minutes for GitHub to build your site
7. Your site will be available at: **https://Morgen33.github.io/gigachad-xrpl/**

## Important Notes:

- The repository must be **Public** for free GitHub Pages hosting
- It may take a few minutes for the site to be available after enabling
- The URL is case-sensitive: `Morgen33` (capital M)

## If Still Getting 404:

1. Check repository is **Public** (Settings > General > Repository visibility)
2. Make sure Pages is enabled (Settings > Pages)
3. Wait 5-10 minutes after enabling Pages
4. Try accessing: `https://Morgen33.github.io/gigachad-xrpl/index.html` directly

## Test Locally First:

Open `index.html` in your browser to make sure the site works locally before troubleshooting GitHub Pages.
